## 项目定位

本项目面向中国海洋大学低年级计算机学生，构建一套 OS 竞赛入门实验体系。项目以 xv6-riscv 为主线，参考 rCore/uCore 等公开课程资源的组织方式，逐步形成环境配置、系统调用、进程与调度、页表与内存、文件系统、最终集成等主题的 step by step 实验教程、代码框架、测试用例、参考实现、FAQ 和过程记录。

项目目标不是替代完整操作系统课程，而是为刚接触 OS 竞赛的学生提供可复现、可验证、可逐步推进的入门路径。
