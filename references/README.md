# 参考资料与阅读路径

## 目标

本文整理与 OUC xv6 Lab Kit 相关的赛题、xv6 入门、完整 OS 课程和往届作品资料，帮助读者按阶段扩展阅读。

## 适用对象

适用于学生、教师、报告维护者和评审。

## 内容范围

资料分为赛题与提交要求、xv6 入门、完整 OS 课程体系、往届作品与文档形态参考。除 xv6-riscv baseline 外，外部资料只作阅读和课程组织参考，不作为本项目实现来源。

## 阅读路径

第一次学习先看 proj54/proj0 赛题、MIT xv6 课程主页和 xv6 book。完成 Lab1-Lab2 后，可阅读 rCore、uCore、PKE 和往届作品，比较课程组织与实验粒度。

## 结构规范

每条参考应写明用途、什么时候看、是否直接用于本项目。未逐页核对的个人资料应标注“未核对”。

## 语言风格

参考资料说明应克制，不把“参考过”写成“实现来源”。

## 质量标准

正式引用、许可证和 URL 最终以 `docs/final/10_reference_and_license_statement.md` 为准。

## 边界条件

不提交 `external/xv6-riscv/`、raw logs、summary 原件、视频、截图、token、密码或隐私材料。不把 timeout evidence 写成长期稳定性测试。不把 `pgcount`/`memstat` 写成完整内存管理；不把 `fcount`/`fdcount`/`fdinfo` 写成完整文件系统；Lab5 不新增内核机制。
