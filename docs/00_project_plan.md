# 项目计划

## 项目目标

面向中国海洋大学低年级计算机学生，建设一套可阅读、可实践、可测试、可复盘的 OS 竞赛入门实验体系。项目以 xv6-riscv 为主线，围绕系统调用、进程调度、页表与内存、文件系统等核心主题，形成 step by step 教程、实验代码框架、测试用例、参考实现、FAQ 和过程记录。
