# 实验体系总体设计

## 设计原则

- 以 xv6-riscv 为主线，尽量减少初学者在多套代码之间切换的成本。
- 每个实验包含背景、目标、任务、代码入口、测试方式、预期产物和 FAQ。
- 所有实验结果以真实执行为准，未执行内容标注 TODO。
- 参考 rCore/uCore 的课程组织方式，但不复制大段外部资料。
