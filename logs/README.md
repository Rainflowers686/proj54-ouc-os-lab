# logs

This directory is reserved for local command logs generated during real build or test runs.

- Raw build logs are not committed by default.
- Future `make` output from xv6 baseline checks should be written as `logs/xv6-make-YYYYMMDD-HHMMSS.log`.
- Key command summaries can be manually excerpted into `docs/04_test_report.md` or `labs/lab0-env-setup/README.md`.
- Do not fabricate logs, command output, screenshots, or success records.
- Do not store passwords, tokens, private account details, registration materials, or large binary files here.
